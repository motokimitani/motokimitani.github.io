Motoki Mitani / 三谷 元紀

1984年生まれ

製造メーカーに勤務し、海外製品の調達・貿易業務を担当。

個人では、オンラインスクールでプログラミングを習得、Web制作を中心にした案件を承っております。

迅速なレスポンスと丁寧な仕事を心掛け、お客様に満足いただけるよう日々学習し、スキル向上に取り組んでおります。

SKILLS

所有スキル

HTML

コーポレートサイト・LP制作・コーディング代行など対応可能。シンプルかつ、ムダのないコードで丁寧に仕上げていきます。「パソコン版・タブレット版・スマホ版」へのレスポンシブ対応もお受けいたします。

CSS

基本的なWebサイトの装飾・表示対応が可能。レイアウト・サイズ調整、色の設定、ボタンのマウスオーバーでの色変化など、ユーザーにとって機能的で見やすいデザインを心掛けて制作しております。

JavaScript

ライブラリ(jQuery)を使用し、基本的なアクション動作の実装に対応可能。Webサイトのスライド機能、カウント機能、お問い合わせフォームなど対応いたします。

English

英語を使用した案件にも対応可能。業務レベルで通訳や資料翻訳、海外業者との交渉などの経験があります。英語サイト制作・英文サイト翻訳等のご依頼もご相談ください。
